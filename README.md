# another
構文構築ローグライクゲーム。1週間ゲームジャム「Unity 1week」参加作品。
テーマ「もうひとつ」より、「1週間でできるだけローグライクゲームを作る」「順番に忠実に実行される構文をゲームのコアとする」というアイデアから、ゲーム体験を設計しました。

## 使用ツール
ゲームエンジン：Unity
グラフィック制作：Adobe Illustrator
コーディングエージェント：Gemini

## 製作期間
2025.12.21~2025.12.28

## 工夫した点・苦労した点
- 上から順番に実行される「構文」という概念をローグライクゲームに導入した。
- 順番入れ替え、良い効果と悪い効果の組み合わせによる奥深い戦略を実現できた。
- 1週間でゲームを完成させた初めての制作で、制作のテンポがつかめなかった。
- グラフィックやBGM, SEなどを期間内に作り切れなかった。
- UI制作に慣れていなかった。

## 課題点、改善点
- 遊び方が直感的でなく、チュートリアルも親切でないためゲームシステムに慣れるのに時間がかかる
- 画面設計が未熟で、UI導線や視覚的魅力が不足している
- BGM, SEがなく、退屈
- ゲームバランスが良くなく、運によってはどうしてもクリアできない状況に陥る。
